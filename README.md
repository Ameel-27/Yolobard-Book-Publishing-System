# Yolobard---Book-Publishing-System
A system to modernizing the traditional book publishing experience all the way from uploading the manuscripts to making the book live.

#Installation:
1. Install xampp version 8.2.12: https://www.apachefriends.org/download.html
2. Download everything from the repository
3. Make sure it's all inside the yolobard folder
4. Put the folder inside C:\xampp\htdocs\
5. Run XAMPP and turn on Apache and MYSQL services
6. Open Admin panel for MYSQL
7. Create a database named "yolobard" and import the yolobard.sql file provided in the repository
8. Open the frontend using the link "http://localhost/yolobard/frontend"
